# ASCII World

## Description

ASCII World is a real-time camera-to-ASCII art web app that transforms your live camera feed into stylized terminal-inspired visuals. It also includes AI-powered scene analysis, image snapshots, and downloadable ASCII text output for a more interactive and shareable experience.

## ✨ Features

- **Live Camera to ASCII:** Convert your webcam feed into animated ASCII art in real time.
- **Visual Controls:** Adjust font size, brightness, contrast, color mode, and character density instantly.
- **AI Scene Analysis:** Capture the current frame and generate a cyberpunk-style analysis using Gemini.
- **Image Snapshot Export:** Save the current ASCII frame as a `.png`.
- **ASCII Text Export:** Download the rendered ASCII output as a `.txt` file.
- **Responsive HUD UI:** Floating controls, repo link, and themed terminal-style presentation.

## 🖥️ Tech Stack

- **Frontend:** React, TypeScript, Vite, Tailwind CSS utilities
- **Icons/UI:** Lucide React
- **AI:** Google Gemini via `@google/genai`

## ⚙️ Setup Instructions

### Prerequisites

- [Node.js](https://nodejs.org/)
- [npm](https://www.npmjs.com/)
- Gemini API key

### Installation

1. **Clone the Repository**
   ```bash
   git clone https://github.com/rishabhgokhe/ASCII-World.git
   ```

2. **Navigate to the Project**
   ```bash
   cd ASCII-World
   ```

3. **Install Dependencies**
   ```bash
   npm install
   ```

4. **Setup Environment Variables**
   Create a `.env.local` file in the project root and add:
   ```env
   GEMINI_API_KEY=your_gemini_api_key_here
   ```

5. **Run the Development Server**
   ```bash
   npm run dev
   ```

6. **Open the App**
   Visit [http://localhost:3000](http://localhost:3000)
